# 🔮 月読み占い - LINE Bot

四柱推命と数霊を組み合わせた本格鑑定サービス

## ✨ 主な特徴

### 🔮 本格的な命式鑑定

- **四柱推命**: 生年月日から年柱・月柱・日柱を算出し、本質・才能・運命を読み解く
- **数霊**: お名前の漢字画数から固有のエネルギーを鑑定
- **温かみのある文章**: 鑑定士が目の前の人に語りかけるようなトーン

### 💾 Supabase によるスケーラブル設計

- **キャッシング**: 同じ条件の鑑定は高速取得（0.1秒以下）
- **無料から対応**: 無料枠で月100万行まで対応
- **自動スケーリング**: ユーザー増加に応じて自動スケール

### 💳 Stripe 決済連携

- 有料鑑定（恋愛・仕事・財運・今年の運勢）
- Checkout Session による安全な決済
- 決済完了後に自動で鑑定結果を送信

## 🚀 クイックスタート

### 前提条件

- Node.js 14.0 以上
- npm
- Supabase アカウント
- LINE Official Account
- Stripe アカウント

### セットアップ手順

1. **依存関係をインストール**

   ```bash
   npm install
   ```

2. **環境変数を設定**

   ```bash
   cp .env.example .env
   # .env に各認証情報を入力
   ```

3. **サーバーを起動**

   ```bash
   npm run dev
   ```

4. **LINE でテスト**
   ```
   田中太郎 1990-05-15
   ```
