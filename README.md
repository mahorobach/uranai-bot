# 🔮 占いの館 - LINE Bot（感動版 v2.0）

**Claude AI + Supabase による感動的な占い体験**

占いの館は、LINE Messaging API上で動作する最新の AI 占い BOT です。
Claude の自然言語処理と四柱推命・数霊を組み合わせ、ユーザーが「自分のことをわかってもらえた」と感じる鑑定体験を実現します。

## ✨ 主な特徴

### 🤖 Claude AI による感動的な鑑定

- **個別対応**: 生年月日と名前から個人に合わせた文章を AI が生成
- **自然言語**: 機械的でなく、温かみのある鑑定テキスト
- **毎回違う表現**: AI がユーザーの期待を超える体験を提供

### 💾 Supabase によるスケーラブル設計

- **無料から対応**: 無料枠で月100万行まで対応
- **自動スケーリング**: ユーザー増加に応じて自動スケール
- **キャッシング**: 同じ条件の人には高速取得（0.1秒以下）

### 👤 ユーザー管理

- LINE ユーザーの自動識別と履歴管理
- 占い結果の蓄積と検索

### 🔄 効率的なキャッシング

- Claude API 呼び出しコスト削減
- 45,600 以上の生年月日パターンをプリコンピューティング可能
- 2回目以降のリクエストで即座に結果返却

## 🚀 クイックスタート

### 前提条件

- Node.js 14.0 以上
- npm
- Claude API キー
- Supabase アカウント
- LINE Official Account

### セットアップ手順

1. **依存関係をインストール**

   ```bash
   npm install
   ```

2. **[SETUP_v2.md](SETUP_v2.md) を読んで環境変数を設定**

   ```bash
   cp .env.example .env
   # SETUP_v2.md を参照して、各認証情報を入力
   ```

3. **サーバーを起動**

   ```bash
   npm run dev
   ```

4. **LINE で テスト**
   ```
   田中太郎 1990-05-15
   ```
